PhotonicsSim

Laser photonics simulation suite — ABCD optics · Gaussian beams · Nonlinear crystals · OPO design

OpticSim
Beam Propagation
Interactive Gaussian beam tracer with 3D visualisation. ABCD matrix propagation through arbitrary optical systems.
  • Thin lens / thick lens / mirror / crystal elements
  • Multi-wavelength + Sellmeier chromatic dispersion
  • Nonlinear crystal SHG search & apply
  • Spot viewer · divergence · Rayleigh range
Open OpticSim →
 CavitySim
Resonator Design
Laser cavity eigenmode solver. Finds the stable Gaussian mode for arbitrary linear standing-wave cavities.
  • Flat & curved mirrors, thin lenses, gain media
  • Stability diagram g₁g₂ · trace half · Rayleigh range
  • Beam profile w(z) plot
  • Export eigenmode → OpticSim URL handoff
Open CavitySim →
 NonlinearSim
Nonlinear Optics
Phase-matching solver and efficiency calculator for SHG, OPO, and QPM processes in common nonlinear crystals.
  • BBO · KTP · LBO · KDP · PPLN Sellmeier databases
  • Type-I/II PM angles · d_eff(θ) · walk-off
  • Boyd-Kleinman focusing · GVD / GVM · temperature tuning
  • OPO threshold P_th calculation
Open NonlinearSim →
 OPODesign
OPO Design Calculator
Integrated OPO threshold optimiser combining cavity mode design (CavitySim) with Boyd-Kleinman nonlinear optics.
  • DRO / SRO threshold P_th vs L and w₀
  • Embedded mini cavity designer (R₁, R₂, L_cav)
  • Optimal focusing ξ_opt ≈ 1.391 · h(ξ) maximisation
  • KTP · BBO · LBO presets

How it runs

Browser
UI Modules
All four modules above run entirely in your browser. No installation, no server, no account. Every calculation happens on your own machine.
Open any HTML file directly — or use GitHub Pages.
CLI
Command Line
node cli.js '{"fn":"NL.getSHGAngle","args":{...}}'

Clone the repo, run with Node.js. Accepts JSON on stdin or as an argument, returns JSON. 18 physics functions. Zero npm dependencies.
Requires: Node.js installed locally.
MCP
Claude Code Integration
Add mcp-server.js to your Claude Code settings and the physics functions become tools Claude can call directly during a conversation — SHG angles, OPO thresholds, cavity eigenmodes, on demand.
Requires: Node.js + Claude Code, running locally.
